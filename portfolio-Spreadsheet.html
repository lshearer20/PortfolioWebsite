Spreadsheet Application

Summary:

This project is a spreadsheet application allowing for simple arithmetic with correct precedence and cell referencing. This was a solo project for my Object-Oriented Software Principles class. The project supports save, load, cell referencing, and simple arithmetic calculations with correct precedence.

My Role:

The entire project with support from in class guidance and the professor.

Techonologies Used:

  • C# / Winforms
  • Visual Studio
  • N-Unit testing

    • Notable Features:

  • Correct precedence for simple arithmetic
  • Cell referencing
  • Save / Load
  • Redo / Undo
  • Cell color change
  • Demo button to automatically fill up cells
